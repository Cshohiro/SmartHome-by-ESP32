# 部品一覧（BOM）

本システムのハードウェア構成に使用した主要部品一覧。

| 部品 | 型番 / 種類 | 用途 |
|------|-------------|------|
| マイコン | ESP32 DevKit V1 | 制御・Webサーバ・IR信号生成 |
| 赤外線LED | 5mm IR LED 940nm | 家電への赤外線送信 |
| 抵抗 | 100Ω | IR LED 電流制限 |
| IR受信モジュール | VS1838B / HX1838 | リモコン信号の受信・解析 |
| ジャンパ線 | Dupont ケーブル | 配線接続 |
| ブレッドボード | 標準タイプ | 試作回路構築 |
| USB電源 | 5V USB電源 | ESP32 電源供給 |

## 備考
- IR送信は GPIO27 を使用
- IR受信は GPIO14 を使用
- 搬送波周波数は 38kHz
