## SmartHome-by-ESP32
ESP32を用いた赤外線家電制御対応のスマートホームIoTシステム
エアコンと照明の制御が可能。

# 実装内容
- ESP32でWiFi接続  
- 赤外線信号の取得と送信  
- Webブラウザからの操作画面作成  
- SPIFFSにWebファイルを書き込み  
- PlatformIO環境で開発  
- タイマー処理による自動信号送信機能の実装

# 構成
- ESP32 + 赤外線LEDモジュールで家電に信号を送信。　　
- HTMLファイルはSPIFFSに保存し、ESP32内のWebサーバーから配信。　　

# 機能
- エアコン操作：暖房冷房切り替え；温度、風量、風向きの調整；タイマー機能　　
- 照明ON/OFF、常夜灯の切替
