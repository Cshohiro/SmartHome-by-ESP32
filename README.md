## SmartHome-by-ESP32
ESP32を用いた赤外線家電制御対応のスマートホームIoTシステム
エアコンと照明の制御が可能。

# システム構成
- ESP32：メイン制御およびWiFi通信
- 赤外線送信モジュール：家電への信号送信
- SPIFFS：Web操作画面ファイル保存
- Webブラウザ：ユーザー操作インターフェース

ESP32上でWebサーバーを動作させ、スマートフォンからアクセスして操作を行う。  
受信した操作指示に応じて赤外線信号を送信し、家電を制御する。

# 実装内容
- ESP32でWiFi接続  
- 赤外線信号の取得と送信  
- Webブラウザからの操作画面作成  
- SPIFFSにWebファイルを書き込み  
- PlatformIO環境で開発  
- タイマー処理による自動信号送信機能の実装

# 構成
- ESP32 + 赤外線LEDモジュールで家電に信号を送信。　　
- HTMLファイルはSPIFFSに保存し、ESP32内のWebサーバーから配信。　　

# 機能
- エアコン操作：暖房冷房切り替え；温度、風量、風向きの調整；タイマー機能　　
- 照明ON/OFF、常夜灯の切替
