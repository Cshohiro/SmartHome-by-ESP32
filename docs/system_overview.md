# システム概要

## 目的
ESP32 を用いて、赤外線リモコン家電（エアコン／照明）を Web UI から操作する。

## システム構成
- スマートフォン／PC：Web UI で操作
- WiFi：同一ネットワーク内アクセス
- ESP32：WebServer + API + IR送信
- 赤外線LED：家電へ信号送信

## データフロー
1. ブラウザで操作（例：暖房／冷房／風量／オフタイマー）
2. ESP32 の HTTP API にリクエスト送信
3. ESP32 が IR 信号を生成して送信
4. 状態は `/api/ac3_state` で取得して UI に反映

## 対応状況
- 3F エアコン：対応（暖房/冷房、温度、風量、停止、オフタイマー）
- 照明：対応（全灯点灯、常夜灯、消灯）
- 2F エアコン：未対応（ページのみ）
